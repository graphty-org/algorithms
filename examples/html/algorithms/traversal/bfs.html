Breadth-First Search (BFS) Explained

BFS explores level by level, like ripples in water.

It visits all neighbors at distance 1, then all at distance 2, and so on.

Key Difference from DFS:

BFS uses a queue (first in, first out) instead of a stack. This ensures we explore all nodes at the current level before moving to the next level.
