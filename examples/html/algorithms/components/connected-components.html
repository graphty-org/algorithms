Connected Components Explained

Connected components are groups of nodes that can reach each other.

If you can walk from node A to node B following edges, they're in the same component.

How Connected Components Work:

The algorithm uses BFS (Breadth-First Search) to explore the graph:

  • Start from an unvisited node and begin a new component
  • Use BFS to visit all nodes reachable from the starting node
  • All visited nodes belong to the same component
  • Repeat with the next unvisited node until all nodes are processed

This efficiently identifies all separate groups in the graph.
