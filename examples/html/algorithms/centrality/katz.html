Katz Centrality Explained

Like PageRank's fair cousin - everyone gets a base score plus influence from popular neighbors!

Katz Centrality gives each node a baseline importance, then adds influence based on how well-connected their neighbors are. Unlike eigenvector centrality, isolated nodes still get some score.

Key Concept:

Katz Centrality solves a key limitation of eigenvector centrality: what happens to nodes with no highly central neighbors? Every node gets a base score (beta), plus additional influence (alpha) from their neighbors' centrality.

This ensures fair treatment of all nodes while still rewarding good connections.
