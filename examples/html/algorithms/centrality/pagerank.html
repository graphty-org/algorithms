PageRank Algorithm Explained

PageRank measures the importance of nodes based on their connections.

Important nodes tend to be connected to other important nodes.

How PageRank Works:

PageRank simulates a random web surfer who:

  • Follows links from page to page with probability = damping factor
  • Jumps to a random page with probability = 1 - damping factor
  • The PageRank score represents how often the surfer visits each page

The algorithm iterates until the scores converge to stable values.
