About This Project

These interactive examples demonstrate the algorithms implemented in the @graphty/algorithms TypeScript library. Each example provides:

  • Interactive visualization with SVG graphics
  • Step-by-step algorithm execution
  • Parameter adjustment and different graph types
  • Educational information about complexity and applications
  • Real-time results and performance metrics
